Kunyx System About GOMU

This is the simplified free version of Kunyx System which is suitable for using in small shops.
To use this version of Kunyx System you have to login with the user name and password (test/test). In this version the user name and password cannot be changed. This version is suitable for shops with only one point-of-sale. The hardware that you need have a POS, thermal printer,a bar-code reader and a customer display. The software elements built into Kunyx database, web-server, POS application and a simplified back-office for managing basic data. When the user logs in the navigation bar appears at the very top of this web-app (Help, VAT, Group, Product, Shop info, Data backup).
A more advanced version of Kunyx System has much more features and for example can add new users and change passwords. This version is charged on an annual subscription basis. The adventage of the advanced version is that it has lot of useful back-office functionalities for managing data like stock, sales tax, sales slip and counter-party data. With the advanced version, support is provided by the Kunyx System team for the subscription period of time by email and phone. This version can use more than one point-of-sale in the same shop with different operators by using user management to add further users.

Login menu:


The successfully logged-in user can start to type in VAT, group, product and shop details and save data using Add or Update buttons. The data entered will be saved into the database of Kunyx System. This version is for single user only and this is the "test" user with password "test".

Shop info menu:


By clicking on the Shop data menu at the navigation bar you can navigate to the page where shop details are saved. Filling up the input fields with shop details and clicking on the Update button will save the data you have entered into Kunyx System's database. Important! - The shop details will appear on the receipt and therefore must be correct and this is entirely the responsibility of the person who running the shop.

VAT menu:


Clicking on this menu will take you to the VAT management page.
All products must have tax rates and as consequence of this is that before specifying the product the VAT rate must be specified. The current standard VAT rate in the UK is 20% (most goods and services) but there are reduced rate 5% and zero rate 0%. When filling up the form for submitting Tax rate the rule is that VAT id can be any character but must be unique. One scenario could be: A for 0%, B for 5% and C for 20% but bear in mind that after attaching VAT to some product and starting to use it in POS and making sales the value-added tax will be calculated and saved in database. The saved calculated value-added tax can be queried for any specified period provided in the last field using the Find button. The last input field is optional and can be used for providing extra information, it can be entirely ignored when using Add/Update. It is important to remember that the last field has crucial role in finding out how much the is amount for specified VAT.

Group menu:


Using Group is not compulsory, but it makes the product management easier. Just click on the Group menu and provide an ID and description for the group (a product category). Type in a choice of name will properly describe that group. The group ID has to be 5 characters and must be unique.

Product menu:


If you want to add a new product to your stock, just click on the product menu to do so. It is very important fact that this action must be preceded by providing the right VAT and proper Group. As mentioned before VAT is compulsory for every product and to have group for product is just recommendation. The other required fields on this form are short name, long name, bar-code and the price of the product. the short name will appear on the receipt. The long name is the name of the product. Bar-code or ID is the product's bar-code or unique ID. Price is the product's gross price. By clicking on the Add button the product details will be saved to the database.
